## Generation Preferences

### Preferred Output Formats
1. **PDF** - Primary, ATS-optimized
2. **HTML** - Web version, responsive
3. **DOCX** - Editable when requested

### Template Preferences
- Font: Calibri 11pt for body, 14pt for headers
- Margins: 1 inch all sides
- Length: 1-2 pages maximum
- Color: Black and white (ATS safe)

## Quality Patterns

### What Works
- Single-column layout
- Standard section headers
- Bullet points with metrics
- Keyword optimization

### What Doesn't
- Tables or columns
- Graphics/charts
- Headers with contact info
- Uncommon fonts
