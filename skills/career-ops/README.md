# Career-Ops Global Skill

This skill allows you to run career-ops commands from any directory.

## Usage

From anywhere, run:
```
/career-ops              → Show command menu
/career-ops {JD or URL} → Evaluate job and generate PDF
/career-ops scan         → Scan job portals
/career-ops tracker      → View application tracker
```

## Files

- Skill definition: `~/.claude/skills/career-ops/SKILL.md`
- Career-Ops installation: `/Users/tanvir/career-ops/`

## To customize career-ops

Edit files in `/Users/tanvir/career-ops/`:
- `cv.md` - Your CV
- `config/profile.yml` - Your profile
- `portals.yml` - Target companies
- `modes/_profile.md` - Your personalization
